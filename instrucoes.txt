📋 INSTRUCTIONS FOR RUNNING YOUR PORTFOLIO

📂 LOCATION:
~/portfolio_website/

📝 MAIN FILES:
- index.html      -> Main website file (32KB)
- styles.css      -> All visual styles (24KB)
- script.js       -> Interactive features (15KB)
- README.md       -> Complete documentation
- run_server.py   -> Python local server script

🚀 TO RUN LOCALLY:

Method 1 - Python Server (Recommended):
1. Open Terminal
2. Navigate to portfolio_website:
   cd ~/portfolio_website

3. Run the server:
   python3 run_server.py

4. Open your browser and go to:
   http://localhost:8000

Method 2 - Direct File:
1. Navigate to ~/portfolio_website/
2. Double-click on index.html
3. It will open in your default browser

Method 3 - VS Code Live Server:
1. Open the folder in VS Code
2. Install "Live Server" extension
3. Right-click index.html -> "Open with Live Server"
