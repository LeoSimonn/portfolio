PowerNutri Evolution

A full nutritional management system: diets, recipes, tracking, and a health analytics dashboard.

GitHub

Overview

The product began as ZapNutri — nutrition follow-up through WhatsApp, with automation and AI to personalize guidance. It grew into PowerNutri Evolution: the same vision, now as a full web platform to organize nutrition end to end — from planning to history — with clear data and a smooth experience. The earlier ZapNutri repository documents that first iteration.

Stack

  • React, Node.js
  • PostgreSQL
  • Supabase, OpenAI / AI flows (earlier ZapNutri iteration)
  • WhatsApp integration (earlier iteration)
